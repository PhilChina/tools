### git上传文件到github上
   1. 在github上  new repository
   2. 在本地新建文件夹，git clone *****.git (从github上黏贴的地址)，就能把项目clone到本地
   3. 将上传的文件复制到clone的项目的目录下
   4. git add --all
   5. git commit -m "更新的描述"
   6. git push
